TIL: Today I Learnt

FiraCode

github.com/tonsky/FiraCode

FiraCode is a free monospaced font containing ligatures for common programming multi-character combinations.

Invidious

github.com/iv-org/invidious

Invidious is a front-end for YouTube that doesn't track you (and does other cool stuff).

Learn X in Y minutes

learnxinyminutes.com

Learn X in Y minutes is a collection of short lessons in various programming languages.

Colemak

colemak.com

Colemak is a modern alternative to the QWERTY and Dvorak layouts, designed for efficient and ergonomic touch typing in English.

sub_module

nginx.org/en/docs/http/ngx_http_sub_module

The ngx_http_sub_module module is a filter that modifies a response by replacing one specified string by another. tl;dr its regex find and replace for things served by nginx.

moar

github.com/walles/moar

Moar is a pager. It reads and displays UTF-8 encoded text from files or pipelines.

thefuck

github.com/nvbn/thefuck

The Fuck is a CLI application that corrects errors in previous console commands. Think git staus autocorrecting to git status.
